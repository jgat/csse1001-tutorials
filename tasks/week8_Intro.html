Week 8 Tutorial: Inheritance

In the previous two weeks we have learnt how to define a new class, as well as some general pointers on designing classes. We wrote a Student class which defines methods for getting, among other things, pieces of information about that Student.

This model for a student can be adapted to modelling employees in a business. You have been provided with an Employee class (see the course website) that implents some basic features that may be useful when modelling an employee.

Your task is to write a Manager class that inherits from Employee that adds or changes the following pieces of functionality:

  • Override the get_vac_days method so that the number of vacation days is 10
  • Create a get_bonus method that returns the manager's bonus
  • Override the get_annual_pay method so that it takes the manager's bonus into account
