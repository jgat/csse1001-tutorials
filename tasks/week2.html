Hello, World!

The purpose of this task is to become familiar with writing, saving, and running programs from IDLE. Write a program which, when run, displays the text Hello, World!. Then, close IDLE, and navigate the filesystem (in Windows Explorer or equivalent) to find the program you just wrote, open it in IDLE, and run it.

Then, modify the program to first ask the user what their name is (using raw_input), and then display a greeting using their name. For example (the blue text is output from the program, the black text is input from the user): 

>>> ================================ RESTART ================================
>>>
What is your name? Sir Lancelot of Camelot
Hello, Sir Lancelot of Camelot!
>>>
Hint:
  • Create a Python source code file by selecting "New" from the File menu in IDLE, or using the Ctrl-N shortcut.
  • Save the file in a location where you will be able to find it. On the lab machines, the H: drive is your personal network drive, which can be accessed from any EAIT machine. Files saved to the C: drive might not be accessible after logging off.
  • By default, IDLE will not add a file extension to your files. You must manually add .py to the end of the filename in order to keep the syntax highlighting.

Alice's Addition Academy

Your 5-year old cousin Alice is learning addition, and you have decided to write a program she can use to practice with. The program will generate two random 2-digit integers, and ask the user what the sum of the two integers is, and display whether or not the user is correct. If the user is incorrect, also display the correct answer.

Some example interactions with the program is as follows: 

>>> ================================ RESTART ================================
>>>
70 + 17 = 87
Correct!
>>> ================================ RESTART ================================
>>>
99 + 56 = 156
No, the correct answer is 155
>>>
Hint:

To generate random integers, first put import random at the top of the file, then use random.randint(10, 99) to generate a random 2-digit integer. 

>>> import random
>>> x = random.randint(10, 99)
>>> x
12

You can also type >>> help(random.randint) at the prompt to get more information. Type help(random) or dir(random) to find more functions in the random module, or find online documentation.
