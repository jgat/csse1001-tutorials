Week 5: Using Enumerate

it is often useful for a for loop to be able to access both the value of some element in a sequence and its index at the same time. We can do this using the enumerate function.

enumerate generates a sequence of tuples, where the first item in the tuple is the index of the current in the sequence and the second item is its value. A for loop can the iterate over both items at once.

Example: 
 for index, item in enumerate('abc'):

Week 11 (Recursion): Alphabetic Words

Comment: This task was originally going to be a warm-up towards the file-traversal one, but the task statement is a little too complicated.

This task is to determine how two words should compare in an alphabetic ordering, without distinguishing between lowercase and uppercase. Specifically, write a function compare(a, b) which takes two strings, and returns -1 if a occurs before b in alphabetic order, 1 if b occurs before a, and 0 if they are the same word (up to capitalisation). The alphabetic ordering of two words is defined as follows:

  • If both strings are empty, then they are the same.
  • If just one of the strings is empty, then it occurs before the other.
  • If the first letter of each word is different, then the ordering of those letters determines the ordering of the words.
  • If the first letter of each word is the same, then compare the remaining words after removing the first letter from each.

For example: 

>>> compare('coconut', 'Cocoon')
-1
>>> compare('swordfish', 'sword')
1
>>> compare('AppLE', 'aPple')
0

It is safe to assume that both inputs contain only letters.

Recursion: List Sorting

A bit too hard for a warm-up task.

Sorting a list of data is a common task in many applications of computing. There are many different methods (or algorithms) for sorting data, one such method, called insertion sort is as follows: first find the smallest element, 'remove' it from the list, then sort the remaining elements, and insert the smallest element back in front of the list.

Write a recursive function sort which takes a list, and returns a sorted version of the list using this method.

Hint:

It may help to use the min function and list.index method, as well as list slicing. 

>>> nums = [5, 2, 4, 1, 3]
>>> min(nums)
1
>>> nums.index(1)
3
>>> nums[3] == 1
True
>>> nums[:3] + nums[4:]
[5, 2, 4, 3]
